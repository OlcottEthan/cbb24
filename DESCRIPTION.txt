Title: Package of College Basketball games from 2023 with helpful functions for analysis
Version: 0.0.1.0000
Authors@R: 
    person("Ethan", "Olcott", , "ethanolcott99@gmail.com", role = c("aut", "cre"),

Description: The package contains the formatted data set of College Basketball games
    from the 2023-2024 season, occuring in the year 2023. It contains functions that
    filter by team, give statistics by team, and a function that computes a tibble
    with all teams win percentage, percentage of home games, and number of games played.
